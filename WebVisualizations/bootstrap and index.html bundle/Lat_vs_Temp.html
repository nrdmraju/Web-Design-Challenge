City Latitude vs Max Temperature

This graph's purpose is to show the change in the max temperature based off of the latitude.
When following the data points on the graph we can see that there is an upward slope in temperature until
0 degrees latitude where there begins a downward slope in temperature. As we can see based on that trend,
as the latitude increases there is a downward slope showing that the temperature does decrease.
