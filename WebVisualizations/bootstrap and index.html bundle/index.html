Weather Data Plots

Summary

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accompolish this analysis, we first pulled data from open weather map API to assemble a Data Set of over 500 cities. After assembling the Data Set, we used matplotlib to plot various aspects of weather vs latitude. Factors we looked at include: temperature, cloudiness, humidity, and wind speed. This site includes the source data and visualizations created for the analysis, as well as their explanations and descriptions
