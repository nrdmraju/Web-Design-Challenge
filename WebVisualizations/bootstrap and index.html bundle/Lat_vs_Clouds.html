City Latitude vs Cloudiness

This graph's purpose is to show the change in the cloudiness based off of the latitude.
When following the data points on the graph we can see that at around -58 degrees latitude is where clouds
starts appearing at around 12% cloudiness. As the latitude approaches 25 degress we can see that the concentration
of the data points showing that the amount of clouds is around 96-98%. As we can see based on that trend, as the latitude
increases there is not a linear increase in clouds which concludes that there is not a strong correlation between latitude and cloudiness.
